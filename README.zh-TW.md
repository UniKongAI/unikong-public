# Whim Windows 版

Whim 是您團隊專用的安全通訊軟體。這個頁面是讓您在 Windows 上下載並安裝它的地方。

---

## 第一步 — 下載

大部分的人只需要這個安裝程式：

### ▶ [下載 Windows 版 Whim](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim-setup.exe)

> **使用的是由 IT 團隊管理的公司電腦嗎？** 他們可能會比較喜歡用
> [封裝版本（.msi）](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim.msi)
> 來在整間公司一起安裝。

## 第二步 — 安裝

1. 對著您下載好的檔案點兩下（雙擊）。
2. 如果 Windows 跳出一個藍色的 **「Windows 已保護您的電腦」（Windows protected your PC）** 視窗，請點選 **更多資訊（More info）**，然後點 **仍要執行（Run anyway）**。（剛發佈的新軟體出現這個提示是正常的。）
3. 當 Windows 詢問是否允許時，請點選 **是（Yes）**。
4. 依照畫面上的步驟操作。安裝完成後，Whim 會自己開啟。

## 第三步 — 登入並開始使用

1. 從您的開始功能表（Start menu）或桌面捷徑開啟 **Whim**。
2. 用您公司使用的帳號登入（您的公司電子郵件或單一登入）。
3. 這樣就完成了 — 您已經可以開始和團隊傳訊息了。

> **不確定要怎麼登入，或要使用哪一個工作區嗎？** 請詢問您的 IT 人員或辦公室管理員。

---

## 保持在最新版本

Whim 會自動更新。您也可以隨時在軟體裡面檢查：**設定 → 檢查更新（Settings → Check for updates）**。

## 移除 Whim

開啟 Windows 的 **設定 → 應用程式 → 已安裝的應用程式（Settings → Apps → Installed apps）**（或 **新增或移除程式（Add or remove programs）**），找到 **Whim**，然後選擇 **解除安裝（Uninstall）**。

## 需要協助嗎？

請聯絡您的 IT 人員或辦公室管理員，或前往 **[unikong.ai](https://unikong.ai)**。
