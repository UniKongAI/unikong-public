# ទាញយក Whim

Whim គឺជាកម្មវិធីផ្ញើសារដ៏មានសុវត្ថិភាពសម្រាប់ក្រុមការងាររបស់អ្នក។ សូមជ្រើសរើសឧបករណ៍របស់អ្នកខាងក្រោម ដើម្បីទាញយកវា។

---

## 🪟 Windows

### ▶ [ទាញយក Whim សម្រាប់ Windows](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim-setup.exe)

> **កំពុងប្រើកុំព្យូទ័រការងារដែលគ្រប់គ្រងដោយក្រុមផ្នែក IT មែនទេ?** ពួកគេប្រហែលជាចូលចិត្ត [កំណែដែលបានវេចខ្ចប់ (.msi)](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim.msi) ច្រើនជាង សម្រាប់ដំឡើងនៅទូទាំងក្រុមហ៊ុនទាំងមូល។

**ដើម្បីដំឡើង៖** សូមចុចពីរដង (double-click) លើឯកសារដែលអ្នកបានទាញយក។ ប្រសិនបើ Windows បង្ហាញប្រអប់ពណ៌ខៀវ **"Windows protected your PC"** (Windows បានការពារកុំព្យូទ័ររបស់អ្នក) សូមចុច **More info** (ព័ត៌មានបន្ថែម) រួចចុច **Run anyway** (ដំណើរការទោះយ៉ាងណាក៏ដោយ)។ សូមចុច **Yes** (បាទ/ចាស) នៅពេលវាសុំការអនុញ្ញាត រួចធ្វើតាមជំហានបន្ត — Whim នឹងបើកដោយខ្លួនឯងនៅពេលដំឡើងរួច ហើយវានឹងធ្វើបច្ចុប្បន្នភាពខ្លួនឯងដោយស្វ័យប្រវត្តិ។
