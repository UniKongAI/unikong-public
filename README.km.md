# Whim សម្រាប់ Windows

Whim គឺជាកម្មវិធីផ្ញើសារដ៏មានសុវត្ថិភាពសម្រាប់ក្រុមរបស់អ្នក។ ទំព័រនេះគឺជាកន្លែងដែលអ្នកទាញយក និងដំឡើងវានៅលើ Windows។

---

## ជំហានទី ១ — ទាញយក

មនុស្សភាគច្រើនត្រូវការតែកម្មវិធីដំឡើងប៉ុណ្ណោះ៖

### ▶ [ទាញយក Whim សម្រាប់ Windows](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim-setup.exe)

> **កំពុងប្រើកុំព្យូទ័រការងារដែលគ្រប់គ្រងដោយក្រុម IT មែនទេ?** ពួកគេប្រហែលជាចូលចិត្ត
> [កំណែដែលបានវេចខ្ចប់ (.msi)](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim.msi)
> ដើម្បីដំឡើងនៅទូទាំងក្រុមហ៊ុនទាំងមូល។

## ជំហានទី ២ — ដំឡើង

1. ចុចពីរដងលើឯកសារដែលអ្នកបានទាញយក។
2. ប្រសិនបើ Windows បង្ហាញប្រអប់ពណ៌ខៀវ **"Windows protected your PC"** សូមចុច **ព័ត៌មានបន្ថែម (More info)** បន្ទាប់មកចុច **ដំណើរការទោះយ៉ាងណាក៏ដោយ (Run anyway)**។ (នេះជារឿងធម្មតាសម្រាប់កម្មវិធីដែលទើបតែបានចេញផ្សាយថ្មី។)
3. ចុច **បាទ/ចាស (Yes)** នៅពេល Windows សុំការអនុញ្ញាត។
4. ធ្វើតាមជំហានដែលបង្ហាញនៅលើអេក្រង់។ Whim នឹងបើកដោយខ្លួនឯងនៅពេលដែលវាបញ្ចប់។

## ជំហានទី ៣ — ចូលគណនី និងចាប់ផ្តើម

1. បើក **Whim** ពីម៉ឺនុយ Start (Start menu) ឬផ្លូវកាត់នៅលើផ្ទៃតុរបស់អ្នក។
2. ចូលគណនីដោយប្រើគណនីដែលកន្លែងធ្វើការរបស់អ្នកប្រើ (អ៊ីមែលការងារ ឬការចូលគណនីតែម្តង single sign-on)។
3. ប៉ុណ្ណឹងហើយ — អ្នករួចរាល់ក្នុងការផ្ញើសារទៅកាន់ក្រុមរបស់អ្នកហើយ។

> **មិនច្បាស់ពីរបៀបចូលគណនី ឬគួរប្រើ workspace ណាមួយ?** សូមសួរក្រុម IT ឬអ្នកគ្រប់គ្រងការិយាល័យរបស់អ្នក។

## ការរក្សាឱ្យទាន់សម័យ

Whim ធ្វើបច្ចុប្បន្នភាពដោយខ្លួនឯងដោយស្វ័យប្រវត្តិ។ អ្នកក៏អាចពិនិត្យបានគ្រប់ពេលពីខាងក្នុងកម្មវិធីដែរ៖ **ការកំណត់ → ពិនិត្យរកបច្ចុប្បន្នភាព (Settings → Check for updates)**។

## ការដកចេញ Whim

បើក **ការកំណត់ → កម្មវិធី → កម្មវិធីដែលបានដំឡើង (Settings → Apps → Installed apps)** របស់ Windows (ឬ **បន្ថែម ឬដកកម្មវិធីចេញ (Add or remove programs)**) រកមើល **Whim** រួចជ្រើសរើស **លុបចេញ (Uninstall)**។

## ត្រូវការជំនួយ?

សូមទាក់ទងក្រុម IT ឬអ្នកគ្រប់គ្រងការិយាល័យរបស់អ្នក ឬចូលទៅកាន់ **[unikong.ai](https://unikong.ai)**។
