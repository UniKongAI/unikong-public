# Windows 版 Whim

Whim 是您团队的安全通讯应用。在这个页面，您可以把它下载并安装到 Windows 电脑上。

---

## 第 1 步 — 下载

大多数人只需要这个安装程序：

### ▶ [下载 Windows 版 Whim](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim-setup.exe)

> **在由 IT 团队管理的办公电脑上使用？** 他们可能更倾向于使用
> [打包版本（.msi）](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim.msi)
> 在整个公司范围内统一安装。

## 第 2 步 — 安装

1. 双击您下载好的文件。
2. 如果 Windows 弹出一个蓝色的 **"Windows 已保护你的电脑"（Windows protected your PC）** 提示框，请点击 **更多信息（More info）**，然后点击 **仍要运行（Run anyway）**。（对于刚刚发布的新应用，出现这种提示是正常的。）
3. 当 Windows 请求权限时，点击 **是（Yes）**。
4. 按照屏幕上的步骤操作。安装完成后，Whim 会自动打开。

## 第 3 步 — 登录并开始使用

1. 从开始菜单（Start menu）或桌面快捷方式打开 **Whim**。
2. 用您单位使用的账号登录（您的工作邮箱或单点登录）。
3. 大功告成 — 您现在可以给团队发消息了。

> **不确定怎么登录，或者该用哪个工作区？** 请咨询您的 IT 或办公室管理员。

---

## 保持最新版本

Whim 会自动更新。您也可以随时在应用内检查更新：**设置 → 检查更新（Settings → Check for updates）**。

## 卸载 Whim

打开 Windows 的 **设置 → 应用 → 已安装的应用（Settings → Apps → Installed apps）**（或 **添加或删除程序（Add or remove programs）**），找到 **Whim**，然后选择 **卸载（Uninstall）**。

## 需要帮助？

请联系您的 IT 或办公室管理员，或访问 **[unikong.ai](https://unikong.ai)**。
