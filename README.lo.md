# ດາວໂຫຼດ Whim

Whim ແມ່ນແອັບສົ່ງຂໍ້ຄວາມທີ່ປອດໄພສຳລັບທີມຂອງເຈົ້າ. ເລືອກອຸປະກອນຂອງເຈົ້າຢູ່ດ້ານລຸ່ມເພື່ອເອົາມັນມາໃຊ້.

---

## 🪟 Windows

### ▶ [ດາວໂຫຼດ Whim ສຳລັບ Windows](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim-setup.exe)

> **ໃຊ້ຄອມພິວເຕີຂອງບ່ອນເຮັດວຽກທີ່ມີທີມ IT ດູແລບໍ?** ເຂົາເຈົ້າອາດຈະມັກ [ສະບັບແພັກເກັດ (.msi)](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim.msi) ຫຼາຍກວ່າ ສຳລັບການຕິດຕັ້ງໃຫ້ທົ່ວທັງບໍລິສັດ.

**ວິທີຕິດຕັ້ງ:** ກົດສອງເທື່ອ (double-click) ໃສ່ໄຟລ໌ທີ່ດາວໂຫຼດມາ. ຖ້າ Windows ສະແດງກ່ອງສີຟ້າທີ່ຂຽນວ່າ **"Windows ໄດ້ປ້ອງກັນຄອມພິວເຕີຂອງເຈົ້າ" (Windows protected your PC)** ໃຫ້ກົດ **ຂໍ້ມູນເພີ່ມເຕີມ (More info)** ແລ້ວກົດ **ແລ່ນຕໍ່ໄປ (Run anyway)**. ກົດ **ແມ່ນ (Yes)** ເມື່ອມັນຖາມຂໍອະນຸຍາດ, ແລ້ວປະຕິບັດຕາມຂັ້ນຕອນ — Whim ຈະເປີດຂຶ້ນເອງເມື່ອຕິດຕັ້ງສຳເລັດແລ້ວ, ແລະ ຈະອັບເດດຕົວເອງໃຫ້ທັນສະໄໝຢູ່ສະເໝີໂດຍອັດຕະໂນມັດ.

## ການເຂົ້າສູ່ລະບົບ (ໃຊ້ໄດ້ກັບທຸກອຸປະກອນ)

1. ເປີດ **Whim**.
2. ເຂົ້າສູ່ລະບົບດ້ວຍບັນຊີທີ່ບ່ອນເຮັດວຽກຂອງເຈົ້າໃຊ້ (ອີເມວວຽກ ຫຼື ການເຂົ້າສູ່ລະບົບແບບລວມ single sign-on ຂອງເຈົ້າ).
3. ແຄ່ນັ້ນເອງ — ເຈົ້າພ້ອມທີ່ຈະສົ່ງຂໍ້ຄວາມຫາທີມຂອງເຈົ້າແລ້ວ.

> **ບໍ່ແນ່ໃຈວ່າຈະເຂົ້າສູ່ລະບົບແນວໃດ ຫຼື ຈະໃຊ້ workspace ໃດ?** ໃຫ້ຖາມທີມ IT ຫຼື ຜູ້ດູແລຫ້ອງການຂອງເຈົ້າ.
