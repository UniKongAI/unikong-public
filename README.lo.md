# Whim ສຳລັບ Windows

Whim ແມ່ນແອັບສົ່ງຂໍ້ຄວາມທີ່ປອດໄພຂອງທີມເຈົ້າ. ໜ້ານີ້ແມ່ນບ່ອນທີ່ເຈົ້າດາວໂຫຼດ ແລະ ຕິດຕັ້ງມັນໃສ່ Windows.

---

## ຂັ້ນຕອນທີ 1 — ດາວໂຫຼດ

ສ່ວນຫຼາຍຄົນພຽງແຕ່ຕ້ອງການຕົວຕິດຕັ້ງເທົ່ານັ້ນ:

### ▶ [ດາວໂຫຼດ Whim ສຳລັບ Windows](https://github.com/UniKongAI/whim-public/releases/latest/download/whim-setup.exe)

> **ໃຊ້ຄອມພິວເຕີຂອງບ່ອນເຮັດວຽກທີ່ມີທີມ IT ດູແລບໍ?** ເຂົາເຈົ້າອາດຈະມັກ
> [ສະບັບແພັກເກັດ (.msi)](https://github.com/UniKongAI/whim-public/releases/latest/download/whim.msi)
> ສຳລັບການຕິດຕັ້ງໃຫ້ທົ່ວທັງບໍລິສັດ.

## ຂັ້ນຕອນທີ 2 — ຕິດຕັ້ງ

1. ກົດສອງເທື່ອ (double-click) ໃສ່ໄຟລ໌ທີ່ເຈົ້າດາວໂຫຼດມາ.
2. ຖ້າ Windows ສະແດງກ່ອງສີຟ້າທີ່ຂຽນວ່າ **"Windows protected your PC"** ໃຫ້ກົດ **ຂໍ້ມູນເພີ່ມເຕີມ (More info)** ແລ້ວກົດ **ແລ່ນຕໍ່ໄປ (Run anyway)**. (ນີ້ແມ່ນເລື່ອງປົກກະຕິສຳລັບແອັບທີ່ຫາກໍເຜີຍແຜ່ໃໝ່.)
3. ກົດ **ແມ່ນ (Yes)** ເມື່ອ Windows ຖາມຂໍອະນຸຍາດ.
4. ປະຕິບັດຕາມຂັ້ນຕອນທີ່ສະແດງຢູ່ໜ້າຈໍ. Whim ຈະເປີດຂຶ້ນເອງເມື່ອຕິດຕັ້ງສຳເລັດແລ້ວ.

## ຂັ້ນຕອນທີ 3 — ເຂົ້າສູ່ລະບົບ ແລະ ເລີ່ມໃຊ້ງານ

1. ເປີດ **Whim** ຈາກເມນູ Start (Start menu) ຫຼື ທາງລັດຢູ່ໜ້າຈໍເດັສທັອບຂອງເຈົ້າ.
2. ເຂົ້າສູ່ລະບົບດ້ວຍບັນຊີທີ່ບ່ອນເຮັດວຽກຂອງເຈົ້າໃຊ້ (ອີເມວວຽກ ຫຼື ການເຂົ້າສູ່ລະບົບແບບລວມ single sign-on ຂອງເຈົ້າ).
3. ແຄ່ນັ້ນເອງ — ເຈົ້າພ້ອມທີ່ຈະສົ່ງຂໍ້ຄວາມຫາທີມຂອງເຈົ້າແລ້ວ.

> **ບໍ່ແນ່ໃຈວ່າຈະເຂົ້າສູ່ລະບົບແນວໃດ ຫຼື ຈະໃຊ້ workspace ໃດ?** ໃຫ້ຖາມທີມ IT ຫຼື ຜູ້ດູແລຫ້ອງການຂອງເຈົ້າ.

---

## ການອັບເດດໃຫ້ທັນສະໄໝ

Whim ຈະອັບເດດຕົວເອງໂດຍອັດຕະໂນມັດ. ເຈົ້າຍັງສາມາດກວດເບິ່ງໄດ້ທຸກເວລາຈາກພາຍໃນແອັບ: **Settings → Check for updates** (ການຕັ້ງຄ່າ → ກວດຫາການອັບເດດ).

## ການລົບ Whim ອອກ

ເປີດ Windows **Settings → Apps → Installed apps** (ການຕັ້ງຄ່າ → ແອັບ → ແອັບທີ່ຕິດຕັ້ງແລ້ວ) (ຫຼື **Add or remove programs** (ເພີ່ມ ຫຼື ລົບໂປຣແກຣມ)) ຊອກຫາ **Whim** ແລ້ວເລືອກ **Uninstall** (ຖອນການຕິດຕັ້ງ).
