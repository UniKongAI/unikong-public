# Windows အတွက် Whim

Whim သည် သင့်အဖွဲ့၏ လုံခြုံသော စာပို့စနစ် (messaging) အက်ပ်ဖြစ်ပါသည်။ ဤစာမျက်နှာသည် ၎င်းကို Windows တွင် ဒေါင်းလုဒ်ဆွဲ၍ ထည့်သွင်းရန် (install) နေရာဖြစ်ပါသည်။

---

## အဆင့် ၁ — ဒေါင်းလုဒ်ဆွဲခြင်း

လူအများစုအတွက် install လုပ်စရာဖိုင် (installer) တစ်ခုသာ လိုအပ်ပါသည်-

### ▶ [Windows အတွက် Whim ကို ဒေါင်းလုဒ်ဆွဲမည်](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim-setup.exe)

> **IT အဖွဲ့က စီမံခန့်ခွဲထားသော အလုပ်သုံးကွန်ပျူတာ ဖြစ်ပါသလား။** ၎င်းတို့သည် ကုမ္ပဏီတစ်ခုလုံးတွင် တစ်ပြိုင်နက် ထည့်သွင်းနိုင်ရန်အတွက်
> [အထုပ်ပြုလုပ်ထားသော ဗားရှင်း (.msi)](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim.msi)
> ကို ပိုနှစ်သက်ကောင်း နှစ်သက်ကြပါမည်။

## အဆင့် ၂ — ထည့်သွင်းခြင်း (Install)

1. သင်ဒေါင်းလုဒ်ဆွဲထားသော ဖိုင်ကို ကလစ်နှစ်ချက်နှိပ်ပါ။
2. Windows က အပြာရောင်အကွက်ဖြင့် **"Windows protected your PC"** ဟု ပြသပါက **More info** ကိုနှိပ်ပြီး၊ ထို့နောက် **Run anyway** ကိုနှိပ်ပါ။ (ဤသည်မှာ အသစ်ထုတ်ဝေထားသော အက်ပ်များအတွက် ပုံမှန်ဖြစ်ပါသည်။)
3. Windows က ခွင့်ပြုချက်တောင်းသောအခါ **Yes** ကိုနှိပ်ပါ။
4. မျက်နှာပြင်ပေါ်ရှိ အဆင့်များအတိုင်း လုပ်ဆောင်ပါ။ ပြီးဆုံးသွားသောအခါ Whim သည် သူ့အလိုလို ပွင့်လာပါမည်။

## အဆင့် ၃ — အကောင့်ဝင်၍ စတင်ခြင်း

1. သင့် Start menu (Start menu) သို့မဟုတ် desktop တွင်ရှိသော shortcut မှ **Whim** ကို ဖွင့်ပါ။
2. သင့်အလုပ်ခွင်မှ အသုံးပြုသော အကောင့် (သင့်အလုပ်သုံး အီးမေးလ် သို့မဟုတ် single sign-on) ဖြင့် အကောင့်ဝင်ပါ။
3. ဒါပဲ — သင့်အဖွဲ့ထံ စာပို့ရန် အသင့်ဖြစ်ပါပြီ။

> **အကောင့်ဘယ်လိုဝင်ရမည် မသေချာဘူးလား၊ ဒါမှမဟုတ် "workspace" ဘယ်ဟာသုံးရမည် မသိဘူးလား။** သင့် IT သို့မဟုတ် ရုံးအုပ်ချုပ်ရေးမှူးကို မေးပါ။

---

## အမြဲတမ်း နောက်ဆုံးပေါ်ဖြစ်အောင် ထားရှိခြင်း

Whim သည် သူ့အလိုလို အလိုအလျောက် အပ်ဒိတ်လုပ်ပါသည်။ အက်ပ်အတွင်းမှလည်း အချိန်မရွေး စစ်ဆေးနိုင်ပါသည်- **Settings → Check for updates** (Settings → Check for updates)။

## Whim ကို ဖယ်ရှားခြင်း

Windows ၏ **Settings → Apps → Installed apps** (Settings → Apps → Installed apps) (သို့မဟုတ် **Add or remove programs** (Add or remove programs)) ကိုဖွင့်ပြီး၊ **Whim** ကိုရှာကာ **Uninstall** (Uninstall) ကိုရွေးပါ။
