# Whim ကို ဒေါင်းလုဒ်ဆွဲခြင်း

Whim သည် သင့်အဖွဲ့၏ လုံခြုံစိတ်ချရသော စာပို့စနစ် (messaging) အက်ပ်ဖြစ်ပါသည်။ ၎င်းကိုရယူရန် သင်အသုံးပြုသော ကိရိယာကို အောက်တွင် ရွေးချယ်ပါ။

---

## 🪟 Windows

### ▶ [Windows အတွက် Whim ကို ဒေါင်းလုဒ်ဆွဲမည်](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim-setup.exe)

> **IT အဖွဲ့က စီမံခန့်ခွဲထားသော အလုပ်သုံးကွန်ပျူတာ ဖြစ်ပါသလား။** ကုမ္ပဏီတစ်ခုလုံးတွင် ထည့်သွင်းရန်အတွက် ၎င်းတို့သည် [အထုပ်ပြုလုပ်ထားသော ဗားရှင်း (.msi)](https://github.com/UniKongAI/unikong-public/releases/latest/download/whim.msi) ကို ပိုနှစ်သက်ကောင်း နှစ်သက်ကြပါမည်။

**ထည့်သွင်းရန် -** ဒေါင်းလုဒ်ဆွဲထားသော ဖိုင်ကို ကလစ်နှစ်ချက်နှိပ်ပါ။ Windows က အပြာရောင်အကွက်ဖြင့် **"Windows protected your PC" (Windows သည် သင့် PC ကို ကာကွယ်ပေးထားသည်)** ဟု ပြသပါက **More info (နောက်ထပ်အချက်အလက်)** ကိုနှိပ်ပြီး၊ ထို့နောက် **Run anyway (မည်သို့ပင်ဖြစ်စေ လုပ်ဆောင်မည်)** ကိုနှိပ်ပါ။ ခွင့်ပြုချက် တောင်းသောအခါ **Yes (ဟုတ်ကဲ့)** ကိုနှိပ်ပြီး၊ အဆင့်များအတိုင်း လုပ်ဆောင်ပါ — ပြီးဆုံးသွားသောအခါ Whim သည် သူ့အလိုလို ပွင့်လာပြီး၊ နောက်ဆုံးပေါ်ဖြစ်အောင် အလိုအလျောက် အပ်ဒိတ်လုပ်ပေးနေပါမည်။

## 🤖 Android

### ▶ [TestFlight တွင် Whim ၏ beta သို့ ဝင်ရောက်ရန်](https://testflight.apple.com/join/YhQ1yvev)

iPhone နှင့် iPad အတွက် Whim သည် Apple ၏ **TestFlight** မှတစ်ဆင့် **beta (စမ်းသပ်ဆဲ ဗားရှင်း)** အဖြစ် ရှိနေပါသည်။ ဦးစွာ App Store မှ အခမဲ့ **TestFlight** အက်ပ်ကို ထည့်သွင်းပါ၊ ထို့နောက် Whim ကိုရယူရန် အထက်ပါ လင့်ခ်ကို ဖွင့်ပါ။ ဗားရှင်းအသစ်များသည် TestFlight မှတစ်ဆင့် ရောက်ရှိလာပါမည်။

## 🍎 Mac

## အကောင့်ဝင်ခြင်း (မည်သည့်ကိရိယာတွင်မဆို)

1. **Whim** ကို ဖွင့်ပါ။
2. သင့်အလုပ်ခွင်မှ အသုံးပြုသော အကောင့် (သင့်အလုပ်သုံး အီးမေးလ် သို့မဟုတ် single sign-on) ဖြင့် အကောင့်ဝင်ပါ။
3. ဒါပဲ — သင့်အဖွဲ့ထံ စာပို့ရန် အသင့်ဖြစ်ပါပြီ။

> **အကောင့်ဘယ်လိုဝင်ရမည် မသေချာဘူးလား၊ ဒါမှမဟုတ် ‘workspace’ ဘယ်ဟာသုံးရမည် မသိဘူးလား။** သင့် IT သို့မဟုတ် ရုံးအုပ်ချုပ်ရေးမှူးကို မေးပါ။
