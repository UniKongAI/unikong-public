# Whim สำหรับ Windows

Whim คือแอปส่งข้อความที่ปลอดภัยสำหรับทีมของคุณ หน้านี้คือที่ที่คุณจะดาวน์โหลดและติดตั้ง Whim บน Windows

---

## ขั้นตอนที่ 1 — ดาวน์โหลด

คนส่วนใหญ่ต้องการแค่ตัวติดตั้งนี้เท่านั้น:

### ▶ [ดาวน์โหลด Whim สำหรับ Windows](https://github.com/UniKongAI/whim-public/releases/latest/download/whim-setup.exe)

> **ใช้คอมพิวเตอร์ของที่ทำงานที่มีทีมไอทีดูแลอยู่ใช่ไหม?** พวกเขาอาจจะชอบ
> [เวอร์ชันแบบแพ็กเกจ (.msi)](https://github.com/UniKongAI/whim-public/releases/latest/download/whim.msi)
> มากกว่า เพราะเอาไว้ติดตั้งให้ทั่วทั้งบริษัทได้

## ขั้นตอนที่ 2 — ติดตั้ง

1. ดับเบิลคลิกที่ไฟล์ที่คุณเพิ่งดาวน์โหลดมา
2. ถ้า Windows ขึ้นกล่องสีน้ำเงินที่เขียนว่า **"Windows ปกป้องเครื่องพีซีของคุณ" (Windows protected your PC)** ให้คลิก **ข้อมูลเพิ่มเติม (More info)** แล้วคลิก **เรียกใช้งานต่อไป (Run anyway)** (เป็นเรื่องปกติสำหรับแอปที่เพิ่งเผยแพร่ใหม่)
3. คลิก **ใช่ (Yes)** เมื่อ Windows ถามขออนุญาต
4. ทำตามขั้นตอนที่ขึ้นบนหน้าจอ เมื่อเสร็จแล้ว Whim จะเปิดขึ้นมาเองโดยอัตโนมัติ

## ขั้นตอนที่ 3 — ลงชื่อเข้าใช้และเริ่มใช้งาน

1. เปิด **Whim** จากเมนู Start (Start menu) หรือไอคอนลัดบนหน้าจอเดสก์ท็อปของคุณ
2. ลงชื่อเข้าใช้ด้วยบัญชีที่ที่ทำงานของคุณใช้ (อีเมลที่ทำงานของคุณ หรือการลงชื่อเข้าใช้แบบครั้งเดียว)
3. เท่านี้ก็เรียบร้อย — คุณพร้อมส่งข้อความถึงทีมของคุณแล้ว

> **ไม่แน่ใจว่าต้องลงชื่อเข้าใช้อย่างไร หรือต้องใช้พื้นที่ทำงานไหน?** สอบถามทีมไอทีหรือผู้ดูแลสำนักงานของคุณได้เลย

---

## การอัปเดตให้เป็นเวอร์ชันล่าสุดอยู่เสมอ

Whim จะอัปเดตตัวเองโดยอัตโนมัติ และคุณยังสามารถตรวจสอบเมื่อไรก็ได้จากในแอป: **การตั้งค่า → ตรวจหาการอัปเดต (Settings → Check for updates)**

## การถอนการติดตั้ง Whim

เปิด **การตั้งค่า → แอป → แอปที่ติดตั้งไว้ (Settings → Apps → Installed apps)** ของ Windows (หรือ **เพิ่มหรือเอาโปรแกรมออก (Add or remove programs)**) จากนั้นมองหา **Whim** แล้วเลือก **ถอนการติดตั้ง (Uninstall)**
